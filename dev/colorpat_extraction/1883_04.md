## Design Pattern

**Diverging Palette**: Two-color diverging with pattern indicating magnitude

**Structure**:
1. Negative extreme: Blue + pattern
2. Negative low: Blue solid
3. Neutral: White/none
4. Positive high: Red/orange + pattern

**Pattern Logic**:
- Pattern (stripes) = High magnitude
- Solid = Low magnitude
- Color (blue vs red/orange) = Direction (negative vs positive)

**Symmetry**:
- Elements 1 and 4 are SYMMETRIC in pattern (both 135° stripes)
- Elements 1 and 4 differ ONLY in color (blue vs orange)
- This creates perfect visual balance for diverging data

## Notes

- **Color + Pattern encoding**: Dual-channel information
  - COLOR encodes direction (blue = negative, orange = positive)
  - PATTERN encodes magnitude (striped = high, solid = low)
- Element 3 (neutral) appears to be NA/transparent in extraction
- Both extremes (1 and 4) use identical pattern (135° stripes, same density)
- Both midpoints (2 and 3) use solid fills
- Used in ggCheysson Guerry vignette for donations quartiles

## Design Insights

This palette demonstrates Cheysson's sophisticated use of DUAL ENCODING:

1. **Color dimension**: Hue contrast (blue vs orange) for categorical distinction (negative vs positive)
2. **Pattern dimension**: Pattern type (stripe vs solid) for ordinal distinction (high vs low magnitude)

This is MORE information-dense than modern diverging palettes which typically use only lightness/saturation to encode magnitude.
