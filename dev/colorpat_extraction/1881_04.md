## Design Pattern

**Grouped/Categorical**: Four distinct hues with solid fills

1. **Element 1**: Warm coral/salmon (#d18781)
2. **Element 2**: Warm yellow/cream (#edd493)
3. **Element 3**: Cool sage green (#7c9a77)
4. **Element 4**: Cool teal-green (#6a8782)

**Color Theory**:
- Alternates warm-warm-cool-cool
- All colors at similar lightness/saturation for equal visual weight
- High mutual distinction for categorical mapping

- All solid fills suggest this was used for a limited color print
- Four-color palette is economical while maintaining clear distinction

## Design Intent

This is a **color-only** grouped palette. The original design did NOT include hatching patterns. When patterns are desired with these colors, they should be added consistently (e.g., all at 45° with same density) rather than varying pattern types, since pattern variation was not part of Cheysson's design for this palette.
