**Pattern Logic**:
- Solid = Low values
- Crosshatch = Medium value (transitional)
- Stripes = High values

## Design Insights

This palette shows an interesting **hybrid strategy**:

1. **Low values (1-3)**: Use COLOR variation (mauve, cream, ?) with solid fills
2. **Transition (4)**: Crosshatch pattern introduces texture
3. **High values (5-7)**: Use PATTERN density variation with constant color

This may reflect:
- Economic printing: Limited to 2-3 colors
- Visual clarity: Solid colors for low (subtle) values, patterns for high (important) values
- Pattern progression is simpler at high end (all diagonal stripes, varying density)
