## Design Pattern

**Sequential Progression**: Single base color (#1e3e69 navy) with increasing pattern density

1. **Element 1**: Diagonal stripes (135°) - simplest pattern
2. **Element 2**: Crosshatch - more complex
3. **Element 3**: Dense crosshatch - most complex

**Historical Context**: This is a monochrome sequential palette designed for printing cost efficiency while maintaining visual distinction through pattern variation alone.

## Notes

- All three elements use the SAME base color (#1e3e69)
- Visual distinction comes ENTIRELY from pattern type and density
- This exemplifies Cheysson's economy: one ink color, three visually distinct categories
- Pattern progression: diagonal → crosshatch → dense crosshatch
- Highly effective for monochrome reproduction
- Modern equivalent would be varying transparency/opacity
